# Code of Conduct

Bosun follows the **Contributor Covenant, version 2.1**.

Read the full text at the canonical source:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

The short version: be considerate and constructive in every project
space — code reviews, issue threads, PR comments, releases. We
maintain a welcoming environment for contributors regardless of
background, experience level, or perspective.

## Reporting

If you experience or observe conduct that runs counter to the
Contributor Covenant, please report it privately to the maintainer:

**jasonmdillingham@gmail.com**

Reports are handled confidentially. The maintainer will acknowledge
within a reasonable window, investigate, and follow the enforcement
ladder described in the Contributor Covenant.

## Scope

This Code of Conduct applies to all bosun project spaces — the
GitHub repository, pull requests, issues, discussions, release
artifacts, and any public communication where the project's name
is used.

It applies to **everyone** participating in those spaces, including
maintainers, contributors, agents being driven by contributors, and
visitors.
